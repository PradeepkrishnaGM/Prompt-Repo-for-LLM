# Contributing

## Contribution types
- New prompt packs (prompt.yaml + tests.jsonl + rubric.yaml + failures.md + output schema)
- Additional test cases (JSONL)
- Benchmark reports
- Bug fixes to tooling (future)

## Requirements for a new prompt pack
A contribution must include:
- `prompt.yaml`
- `tests.jsonl` (minimum 5 cases)
- `rubric.yaml`
- `failures.md` (at least 1 known/anticipated failure mode)
- output JSON Schema under `schemas/`
